


# My personal academic website

This is my own website using [Github pages](https://pages.github.com/) and [Jekyll](https://jekyllrb.com/)

## Feel free to use this template

* clone the project
* install Command Line Tools
* install Ruby
* install Jekyll
* `cd` to the project
* run `jekyll new .` to start

## Technologies used

- Ruby
- Javascript and JSX
- CSS3
- HTML5
- Sass

## Dependencies
- [npm & node.js](https://nodejs.org/en/)
- [React](https://reactjs.org/)
- [RubyGems](https://rubygems.org/)
- [Sass](https://sass-lang.com/)
- [al-folio](https://github.com/alshedivat/al-folio)
